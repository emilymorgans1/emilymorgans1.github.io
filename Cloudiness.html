    Cloudiness

    The purpose of this project was to analayze how weather changes as you get closer to the equator. To accomplish this. analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 50 cities

    After assembling the dataset, we used Matplotlob to plot various aspects of the weather vs. lattitude. Factors we looked at included: temperature, Cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as wll as explanations and descriptions of any trends and correlations witnessed.
